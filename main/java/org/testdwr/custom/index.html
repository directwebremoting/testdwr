Custom Error Pages

This servlet is almost certainly very insecure. It is highly likely that it contains persistent XSS flaws, or other more significant issues. It should not be deployed into an untrusted environment.

The following error URLs are provided by this application

  • HTTP/100: Continue - Tells the browser to carry on sending the request.
  • HTTP/101: Switching Protocols - If we need to upgrade to HTTP 1.1.
  • HTTP/200: OK - The request has succeeded.
  • HTTP/201: Created - (Although we do not create any resources or return a Location header)
  • HTTP/202: Accepted - Intentionally non-committal response
  • HTTP/203: Non-Authoritative - Perhaps response is from a cached copy
  • HTTP/204: No Content - Browser should keep the existing view
  • HTTP/205: Reset Content - Browser should reset the content to load state
  • HTTP/206: Partial Content - Sent in response to a GET with a Range header
  • HTTP/300: Multiple Choices - Page indicates options
  • HTTP/301: Moved Permanently - New URI is this page, or is specified after /
  • HTTP/302: Found - Like 301, but the Location header isn't permanent
  • HTTP/303: See Other - For form submit re-direction
  • HTTP/304: Not Modified - In response to a conditional GET
  • HTTP/305: Use Proxy - Use Location header as proxy
  • HTTP/307: Temporary Redirect - Like 301, but the Location header isn't permanent
  • HTTP/400: Bad Request - The request was malformed
  • HTTP/401: Unauthorized - Use HTTP-Auth
  • HTTP/402: Payment Required - Reserved for future use
  • HTTP/403: Forbidden - Security access denied
  • HTTP/404: Not Found - URL not found
  • HTTP/405: Method Not Allowed - HTTP-Method disallowed
  • HTTP/406: Not Acceptable - Request headers invalid for URI
  • HTTP/407: Proxy Authentication Required - 401 for a proxy
  • HTTP/408: Request Timeout - You can try again later
  • HTTP/409: Conflict - Maybe the user can resolve the conflict
  • HTTP/410: Gone - If you had a bookmark here, delete it
  • HTTP/411: Length Required - Repeat request with a Content-Length header
  • HTTP/412: Precondition Failed - Some headers contained preconditions which failed
  • HTTP/413: Request Entity Too Large - I've got it, but it's too big to give out
  • HTTP/414: Request-URI Too Long - Only likely when POST is converted to a GET
  • HTTP/415: Unsupported Media Type - Requested resource not in requested format
  • HTTP/416: Requested Range Not Satisfiable - Only with Range request header
  • HTTP/417: Expectation Failed - Only with Expect header
  • HTTP/500: Internal Server Error - Generic server failure
  • HTTP/501: Not Implemented - The server can't fulfill the request
  • HTTP/502: Bad Gateway - An upstream proxy failed
  • HTTP/503: Service Unavailable - Under maintenance or overloaded
  • HTTP/504: Gateway Timeout - Upstream proxy timeout
  • HTTP/505: HTTP Version Not Supported -
